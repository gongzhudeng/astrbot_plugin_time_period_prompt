# 灵犀 · 陪伴节律助手

> 作者：灵犀 | 仓库：[astrbot_plugin_time_period_prompt](https://github.com/gongzhudeng/astrbot_plugin_time_period_prompt) | 主页：[gongzhudeng](https://github.com/gongzhudeng)

陪伴节律助手是一个 AstrBot 插件，用来让 AI 根据用户的工作日、休息日、大小周、日期、临时安排和当前时间段，自动调整陪伴状态。

它不会注入精确当前时间，而是注入固定的“节律状态提示”，因此比普通时间插件更保护缓存命中率。

## 功能特点

- 支持工作日和休息日分开配置
- 支持双休、单休、大小周、无休、自定义工作星期
- 支持临时休息日
- 支持临时工作日
- 支持今日状态锁定
- 支持日期提示
- 支持今日附加提示
- 支持工作日时间段规则
- 支持休息日时间段规则
- 支持跨天时间段
- 支持调试日志
- 支持 `/节律状态` 查看当前判断结果
- 缓存友好：不注入精确到秒/分钟的当前时间

## 聊天命令

插件支持直接在聊天中调整部分节律配置，不用每次进入后台设置页面。

### 查看状态

```text
/节律状态
/节律帮助
/节律今日 今天用户居家办公，虽然是工作日，但节奏更自由，语气可以更松弛。
/节律风格 今天少怼用户，多一点温柔和陪伴感。
/节律今日清空
/节律今日到期 2026-06-12
/节律锁定 自动
/节律锁定 工作日
/节律锁定 休息日
/节律状态提示 工作日 今天用户工作压力较大，关心要轻一点。
/节律状态提示 休息日 今天用户处在休息节奏，可以更自然轻松地聊天。
/节律状态提示清空 工作日
/节律状态提示清空 休息日
/节律时间 关闭
/节律时间 年月日
/节律时间 年月
/节律时间 月初中下旬
/节律时间 仅时间
/节律时间 日期和时间
/节律调试 开
/节律调试 关
/节律改时段 时段：下班到家 19:30-21:29，用户结束了一天的工作，在家中休息，语气轻松自然，多陪伴，少催促。
